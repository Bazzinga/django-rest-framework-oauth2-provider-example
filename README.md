django-rest-framework-oauth2-provider
===================
Working example of django-rest-framework using OAuth2 authentication (django-oauth2-provider).

## Install 
Clone this repo, set up a virtualenv:
```console

```

Setup dependencies and database 
```console
make install
make initdb
```

## Run the server 
```console
make server
```

## Tests 
```console
make test 
```
